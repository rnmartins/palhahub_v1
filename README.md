# Palha Italiana

Aplicação React + Vite para apresentar a marca Palha Italiana com uma landing page moderna, responsiva e com animações suaves.

## Tecnologias

- React
- Vite
- Framer Motion

## Como rodar localmente

```bash
npm install
npm run dev
```

## Estrutura principal

- `src/` — componentes, páginas e estilos da aplicação
- `public/` — arquivos estáticos, se necessário
- `images/` — imagens usadas na landing page

## Personalizar

- Edite os componentes em `src/components/` para alterar seções e conteúdo
- Ajuste a identidade visual em `src/styles.css`
- Troque ou adicione imagens na pasta `images/`

## Build para produção

```bash
npm run build
```
